# RecordButton
录音按钮

如何使用

**先在 build.gradle(Project:XXXX) 的 repositories 添加:**

```
allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

**然后在 build.gradle(Module:app) 的 dependencies 添加:**

```
	dependencies {
	        compile 'com.github.xuanu:RecordButton:0.0.1'
	}
```
